Self-Hosted Deployment

Learn how to deploy your Obiya project to production using Obiya-docker

We provide a docker image that you can use to deploy your Obiya project to production. You can use it as a base image and add your own customizations or use it as is.

You will need to use the latest version of the Obiya package 0.5.2-beta.101 or higher
